# TGWatcher

Экстренный доступ к вашим Telegram аккаунтам когда SMS коды заблокированы.

## Что такое TGWatcher?

TGWatcher это Telegram бот который помогает пользователям сохранить доступ к аккаунтам во время проблем с SMS из-за блокировок РКН. Он отслеживает коды входа отправленные на авторизованные устройства и поддерживает резервные сессии активными.

## Зачем это нужно?

Поскольку РКН заблокировал SMS сервисы Telegram в России, пользователи не могут получать коды входа через SMS. Единственный способ войти это коды отправленные на уже авторизованные устройства. TGWatcher решает эту проблему:

- Хранит резервные string session для экстренного доступа
- Отслеживает коды входа от сервиса Telegram
- Автоматически поддерживает сессии активными
- Помогает восстановить доступ после переустановки приложения

## Быстрый старт

Установка одной командой через curl:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/i-execute/TGWatcher/main/Storage/Installation/QuickStart.sh)
```

Установка одной командой через wget:

```bash
bash <(wget -qO- https://raw.githubusercontent.com/i-execute/TGWatcher/main/Storage/Installation/QuickStart.sh)
```

Ручная установка:

```bash
git clone https://github.com/i-execute/TGWatcher.git
cd TGWatcher/Storage/Installation
chmod +x Setuper.sh
./Setuper.sh
```

## Конфигурация

При установке нужно будет указать:

- `BOT_TOKEN` - токен вашего бота от @BotFather
- `OWNER_ID` - ваш Telegram ID
- `API_ID` - с https://my.telegram.org
- `API_HASH` - с https://my.telegram.org

## Использование

Команды администратора:

- `/start` - главное меню со статусом обновлений
- `/menu` - быстрое меню
- `/admin` - панель администратора (только владелец)
- `/packages` - менеджер пакетов
- `/check` - проверить все сессии
- `/backup` - создать файл бэкапа
- `/update` - проверить и установить обновления

Для обычных пользователей:

- `/start` - приветствие и инструкции
- Отправить любую ссылку - симуляция загрузки

## Управление пакетами

1. Отправить string session для установки
2. Отправить файл `.session`
3. Отправить `.zip` архив с сессиями
4. Использовать меню `/packages` для управления

## Управление администраторами

1. Использовать команду `/admin` (только владелец)
2. Нажать Add
3. Отправить ID пользователя
4. Пользователь автоматически разбанится если был забанен

## Как это работает

**Session Keeper**

Каждый слот сессии работает по случайному расписанию:

- 1-10 подключений в день
- Каждое подключение длится 1-30 минут
- Расписание обновляется ежедневно
- Предотвращает таймаут сессии

**Мониторинг кодов**

Бот отслеживает сообщения от 777000:

- Коды входа: `Login code: 12345`
- Веб коды: `login code: AbCdEfGh`
- Все администраторы получают мгновенное уведомление

**Безопасность**

- Автоматически удаляет чувствительные сообщения
- Защита от спама (лимиты 5/10с и 10/60с)
- Автобан спамеров
- Автовыход из групп
- Назначение администраторов только владельцем

## Управление сервисом

```bash
systemctl --user status tgwatcher
systemctl --user restart tgwatcher
systemctl --user stop tgwatcher
journalctl --user -u tgwatcher -f
```

## Решение проблем

Бот не запускается:

```bash
journalctl --user -u tgwatcher -f
```

Обновление не удалось:

```bash
cd ~/.local/tgwatcher
git pull origin main
systemctl --user restart tgwatcher
```

Сбросить всё:

```bash
systemctl --user stop tgwatcher
rm -rf ~/.local/tgwatcher
```

Затем запустить установщик снова.
